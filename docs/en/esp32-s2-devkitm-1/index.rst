ESP32-S2-DevKitM-1
==================

:link_to_translation:`zh_CN:[中文]`

.. note::

    As the ESP32-S2-DevKitM-1 development board offers essentially the same functionality and applications as ESP32-S2-DevKitC-1-N8R2, its production has been discontinued. Developers may use ESP32-S2-DevKitC-1-N8R2 for software and basic function testing. This change does not affect the supply of the ESP32-S2-MINI-2/2U modules used on the board.

ESP32-S2-DevKitM-1 is entry-level development board. Most of the I/O pins on the module are broken out to the pin headers on both sides for easy interfacing. Developers can either connect peripherals with jumper wires or mount ESP32-S2-DevKitM-1 on a breadboard.
