=======================
ESP32-C3-DevKit-RUST-2
=======================

:link_to_translation:`en: [English]`

ESP32-C3-DevKit-RUST-2 开发板使用配置 4 MB SPI flash 的通用型模组 `ESP32-C3-MINI-1 <https://documentation.espressif.com/esp32-c3-mini-1_datasheet_cn.html>`_。该开发板具备完整的 Wi-Fi 和低功耗蓝牙® 功能。
